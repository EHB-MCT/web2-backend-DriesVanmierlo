Kapsalon API documentation

Kapsalon routes

  • GET / - Documentation
  • GET /kapsalons - get all kapsalons
  • GET /kapsalon/61b9df9cd5c8a1b90a90a5ff - get kapsalon with id (from mongodb)
  • GET /kapsalons?id=123456 - Get the kapsalon with kapid 123456
  • POST /saveKapsalon - saves the kapsalon with all data
  • PUT /updateKapsalon/id - updates all the data of the kapsalon with id
  • PUT /rateKapsalon/id - updates only the ratings and latest general rating of kapsalon with id
  • DELETE /deleteKapsalon/id - deletes kapsalon with id

Admin routes

  • GET /admins - get all admins
  • POST /registerAdmin - saves a new admin
  • POST /loginAdmin - checks if admindata is correct
